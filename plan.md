
# Project Plan - Rovo Agency Website

**Goal**: Build a high-end, modern software agency landing page called "Rovo".
**Stack**: React (Vite), TypeScript, Tailwind CSS (CDN).

## Features
- Modern Hero section with gradient text.
- Services grid showing expertise (Web, Cloud, AI, Mobile).
- Portfolio/Work showcase.
- Contact section.
- Fully responsive design.
- Dark theme aesthetic.
